# Aerospace Skills Repository

This repository contains Opencode skills for aerospace-related software development, particularly focused on amateur and student rocketry projects.

## Structure

- `skills/` - Directory containing individual skills
  - `amateur-rocketry/` - Skill for amateur/student rocketry software development
- `openspec/` - SDD (Spec-Driven Development) artifacts (optional, can be ignored for skill usage)
- `.atl/` - Internal directory for skill registry (managed by Opencode)

## Getting Started

To use the skills in this repository with Opencode:

1. Clone this repository to your local machine
2. Ensure Opencode is installed and configured
3. The skills will be automatically discovered from the `skills/` directory

## Available Skills

### Amateur Rocketry

Specialized skill for amateur/student rocketry software development with focus on safety standards, telemetry, and flight software patterns. Use when developing embedded systems for rockets, payloads, or ground support equipment in educational or amateur contexts.

**Triggers**: embedded rocketry, STM32 telemetry, rocket programming, flight software, payload development, amateur rocketry, student rocketry, high-power rocketry, model rocketry, IREC, ESRA, spaceport america cup
